Joining

Overview

Multiple spatial objects can be joined into a single object:

Base.join — Function
join(object₁, object₂, joiner)

Join spatial object object₁ and object₂ with joiner method.

Example

Ω = georef((z=rand(100,100),))

join(Ω, Ω, VariableJoiner())
100×100 RegularGrid{Float64,2}
  variables
    └─z1 (Float64)
    └─z2 (Float64)

Methods

GeoStatsBase.VariableJoiner — Type
VariableJoiner()

Join spatial data with shared domain to produce spatial data with all variables included.
