Covering

Overview

Spatial objects can be covered with spatial regions (e.g. bounding boxes):

GeoStatsBase.cover — Method
cover(object, coverer)

Cover spatial object with coverer method.

Other utility functions are available, which are implemented with the general cover function:

GeoStatsBase.boundbox — Function
boundbox(object)

Return the minimum axis-aligned bounding rectangle of the spatial object.

Notes

Equivalent to cover(object, RectangleCoverer())

Example

R = boundbox(RegularGrid(50, 80))

lowerleft(R), upperright(R)
([0.0, 0.0], [49.0, 79.0])

Methods
