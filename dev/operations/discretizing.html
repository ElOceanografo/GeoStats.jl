Discretizing

Overview

Geometries can be discretized:

GeoStatsBase.discretize — Function
discretize(region, method)

Discretize spatial region with discretization method.

Example

R = Rectangle((0., 0.), (100., 100.))

D = discretize(R, BlockDiscretization((5, 4)))

plot(D)

Methods

GeoStatsBase.BlockDiscretization — Type
BlockDiscretization(dims)

Discretize spatial region into dims blocks.
