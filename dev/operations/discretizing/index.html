Discretizing

Overview

Geometries can be discretized:

GeoStatsBase.discretize — Function
discretize(region, discretizer)

Discretize spatial region with discretizer method.

Example

R = Rectangle((0., 0.), (100., 100.))

D = discretize(R, RegularGridDiscretizer((5, 4)))

plot(D)

Methods

GeoStatsBase.RegularGridDiscretizer — Type
RegularGridDiscretizer(dims)

Discretize spatial region into regular grid of given dimensions dims.
