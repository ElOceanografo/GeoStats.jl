Detrending

Overview

Spatial data can be detrended in place:

GeoStatsBase.detrend! — Function
detrend!(sdata, vars; degree=1)

Detrend the variables vars in the spatial sdata in place, and return the trends.

Alternatively, the trend can be obtained without mutating the input data:

GeoStatsBase.trend — Function
trend(sdata, vars; degree=1)

Return the deterministic spatial trend for the variables vars in the spatial sdata. Approximate the trend with a polynomial of given degree.

Example

# quadratic + noise
r = range(-1,stop=1,length=100)
μ = [x^2 + y^2 for x in r, y in r]
ϵ = 0.1rand(100,100)
𝒟 = georef((z=μ+ϵ,))

ℳ = trend(𝒟, :z, degree=2)

plot(plot(𝒟), plot(ℳ))
