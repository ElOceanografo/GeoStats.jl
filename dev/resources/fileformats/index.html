File formats

The Julia ecosystem provides various packages for reading/writing spatial data from disk.

NetCDF

  • NetCDF.jl - MATLAB-like syntax for reading/writing NetCDF files.
  • NCDatasets.jl - Read/write of NetCDF files and lazy loading.

GRIB

  • GRIB.jl - Read/write GRIB files in Julia.

GDAL

  • GDAL.jl - Julia wrapper for GDAL.
  • ArchGDAL.jl - High-level API for GDAL.

GSLIB

  • GslibIO.jl - Read/write extended GSLIB files in Julia.
